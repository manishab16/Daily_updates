lsb_release -a will tell your distribution codename

apt-cache located in /var/lib/apt/lists
apt-cache search <search term> will search cache for all packages names appropriate to search term
apt-cache show <package name> will show detailed information about the package


$sudo apt-get update

$sudo apt-get upgrade 

sudo apt-get update updates your cache
sudo apt-get upgrade upgrades all software on your system
make sure you update chache before your upgrade
